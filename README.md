# Präsentationen RO
Für jede übung ausser der 0 und C gibt es einen ordner.  
Um eine davon zu öffnen jeweils in der konsole in den ordnet navigieren und `pnpm install && pnpm dev' ausführen. Dann `localhost:3030` im Browser öffnen.  

> [!TIP]
> Das kann eine weile dauern zu laden, nicht wundern.  
> Die Konsole muss auch offen bleiben während die Präsentation läuft
